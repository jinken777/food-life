# Food Life（フードライフ）

Food Life is a gentle world of anthropomorphic food characters.

This project is a growing AI-assisted world where small jobs, quiet kindness, and everyday life matter.

食品キャラクターたちが暮らす、
優しくて少し不思議な街「Food Life」。

小さな仕事や、静かな思いやりを大切にしながら、
AIと人間が一緒に育てている世界です。
